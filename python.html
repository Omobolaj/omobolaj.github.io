Customer Call List – Data Cleaning with Pandas

Tools Used: Python, Jupyter Notebook, Pandas, Regex, DataFrame Transformations

Problem Solved: The customer call list contained formatting inconsistencies, null entries, misplaced delimiters, and inaccurate categorizations which made it unsuitable for communication or analysis.

Solution Provided: I imported the dataset and used Pandas to remove duplicates, clean text fields, standardize phone numbers with regex, split full address into structured components, and encode categorical values to ensure uniformity.

Outcome: The data was fully cleaned and standardized, resulting in a professional, analysis-ready customer list. The cleaned output improved usability for sales/marketing teams and minimized contact errors.

Sales Visualization Project – Interactive Trend Analysis with Matplotlib

Tools Used: Python, Pandas, NumPy, Matplotlib, Jupyter Notebook

Problem Solved: Business stakeholders lacked intuitive insights into how product categories, city distribution, payment methods, and time trends influenced total revenue generation.

Solution Provided: I read, grouped, and visualized sales data using pandas and matplotlib, showcasing key KPIs like highest spending categories, top-performing cities, monthly sales variations, and transaction method share.

Outcome: Visual dashboards simplified pattern detection, enabling sales leaders to prioritize top-selling categories, optimize city-level inventory allocations, and favor preferred customer payment methods accelerating strategic decision-making.

Web Scraping Remote Jobs – Real-Time Listings Extraction with Python

Tools Used: Python, Requests, BeautifulSoup, Pandas, JSON API

Problem Solved: Accessing fresh remote job listings manually from sites like RemoteOK was inefficient and limited search customization.

Solution Provided: Built a Python scraper using the RemoteOK API and BeautifulSoup to automatically extract job data such as titles, companies, locations, tags, and links. Data was stored in a structured CSV for downstream filtering or analysis.

Outcome: Automated retrieval of 90+ job listings in real-time, enabling dynamic dashboards, instant filtering, and seamless integration with job-alert systems for remote tech opportunities.
